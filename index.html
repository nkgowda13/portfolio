Conveyor Automation Project

Conveyor Control System with PLC Logic, OPC UA, and HMI-Style Simulation

Developed an end-to-end conveyor control project in which a box is detected by two sensors, starts conveyor movement on first detection, and stops at the second sensor based on defined control logic. The system links a Unity-based simulation with CODESYS through OPC UA, demonstrating real-time communication, control sequencing, and process visualisation across an industrial-style workflow.

  • Implemented sensor-based control logic for conveyor start/stop sequencing and repeatable system behaviour.
  • Integrated Unity and CODESYS using OPC UA to enable real-time communication between simulation and control logic.
  • Applied industrial automation concepts including PLC-style sequencing, HMI-oriented visualisation, testing, and debugging.
  • Demonstrates practical skills relevant to controls, industrial automation, commissioning support, and system integration roles.
